# data4000-assignment5

Assignment 5 - Conditionals, Loops, Exceptions, and Libraries

This program contains 2 multi-step problems 
1. pos_checkout.py - Point-of-Sale Checkout System
2. inventory_spotcheck.py - Inventory Reorder Analyzer with API Spot Check

Required Installation Command: 
python3 -m pip install requests
Also needed to set up the Student Key for both questions 

How to Run Programs: 
- run each file from the terminal inside the folder
- python pos_checkout.py
- Python inventory_spotcheck.py
